## About Novon
Novon is a free and open source web novel reader built with Flutter. It focuses on speed, privacy, and a smooth reading experience across all platforms.
It uses a community driven extension system that lets you access novels from many sources in one place, You can download chapters, keep your progress saved, and read without needing an internet connection.
Novon is fast, simple to use, and gives you full control over how you read. It is built by readers for readers.
